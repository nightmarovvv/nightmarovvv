## `> whoami`

I break software for a living and automate everything I can get my hands on.

By day, I'm a **QA Automation Engineer** — designing test architectures, building CI/CD pipelines, and making sure nothing ships broken. Python is my weapon of choice: **Playwright** for browser automation, **Pytest + Vedro** for test frameworks, **Allure** for reporting.

By night, I'm deep in **DeFi** — providing liquidity on **Meteora**, trading memecoins, running bots, and teaching others how to do the same. Currently co-running an **LP educational program** on Meteora with a partner.

I've been through the full crypto cycle: spot trading, margin longs & shorts, memecoin degen plays, and now settled into sustainable yield through concentrated liquidity. The common thread? **Automation** — if I'm doing it more than twice, there's a script for it.
